# 宜大美食地圖 — 部署與推廣建議

**Languages:** [English](deployment-strategy.md) · [中文](部署與推廣建議.md)

> 目標：讓國立宜蘭大學學生能透過手機／電腦使用美食地圖與轉盤選擇器。  
> 專案路徑：`python程式期末專案_美食系統`

---

## 第 4 點：APP 還是雲端？

**建議：先做「雲端網頁（PWA 可選）」，不要先做原生 APP。**

| | 雲端網頁 | 原生 APP |
|---|----------|----------|
| **給宜大學生推廣** | 貼連結／QR 即用，iOS／Android 都開 | 要上架或 sideload，門檻高 |
| **維護成本** | 改一次全體更新 | iOS + Android 兩套 |
| **適合你現有架構** | Streamlit → Streamlit Community Cloud／Render／Railway | 要重寫前端 |
| **期末 demo** | 當天就能部署 | 來不及 |

---

## 推薦路線（給宜大學生用）

### 1. 現在（期末）

- 使用 **Streamlit Community Cloud** 免費部署
- 把網址丟系上、社團、LINE 群
- 本機驗證指令：

```powershell
cd python程式期末專案_美食系統
$env:PYTHONPATH = "src"
streamlit run src/streamlit_app.py
```

### 2. 暑假後（用量變大時）

- 遷到 **FastAPI + 靜態前端**
- 轉盤元件（`src/foodmap/wheel.py`）已是 HTML，可重用
- 掛學校子網域，例如 `foodmap.niu.edu.tw`（需向學校資訊單位申請）

### 3. PWA（可選，成本低）

- 加 `manifest.json` + 主畫面圖示
- 手機瀏覽器「加到主畫面」→ 體驗接近 APP
- **仍免 App Store／Play 上架**

### 4. 原生 APP（暫不需要）

等有明確需求再做，例如：

- 推播通知（新店家、營業提醒）
- 離線快取
- 背景定位

目前專案（清單、地圖、轉盤、評分）**雲端網頁已足夠**。

## 部署注意事項

### 資料檔

| 檔案 | 說明 |
|------|------|
| `data/places_cache.json` | 真實 Google Places 快取；部署時需放在雲端可讀路徑，或改讀 repo 內 `sample_restaurants.json` |
| `src/foodmap/data/sample_restaurants.json` | 內建 Mock，無 Key 也可 demo |

### 安全

- **`GOOGLE_MAPS_API_KEY` 不可 commit 進 Git**
- `.env`、快取已在 `.gitignore`
- 公開 repo 前執行：`git status --short --ignored`，確認無金鑰與內部檔

### Streamlit Cloud 設定參考

| 項目 | 建議值 |
|------|--------|
| Main file | `src/streamlit_app.py` |
| Python | 3.10+ |
| 環境變數 | `PYTHONPATH=src`（或於 repo 根目錄調整 import） |
| Secrets | 僅在需要**重新抓** Places 時設定 `GOOGLE_MAPS_API_KEY`；日常查詢讀 JSON 快取即可 |
